# WhatsApp 风格未读角标和标题优化

## 改动概览

针对 `ChatPage.vue` 进行了进一步的 WhatsApp 风格优化，重点是未读角标布局和标题样式。

## 详细改动

### 1. 侧边栏标题样式优化

**文件**: `frontend/src/pages/ChatPage.vue`

**标题 "EdgeChat" 改进**:
- 字体大小：20px → 21px
- 字体粗细：700 → 600（更自然）
- 颜色：#008069（绿色）→ #111b21（深灰黑）
- 字距：增加 -0.3px 的紧凑感
- 背景：白色 → 浅灰色 (#f0f2f5)
- 添加底部边框分隔：1px solid #e9edef

**效果**: 标题更加简洁专业，符合 WhatsApp 的现代风格，不再使用显眼的绿色，而是采用低调的深色。

### 2. 未读角标布局优化

#### 角标位置调整
**之前**: 角标在预览文字的右侧（底部行）
```
[头像] [名字]              [时间]
       [预览文字] [角标]
```

**现在**: 角标在底部行的最右侧，与预览文字分开
```
[头像] [名字]              [时间]
       [预览文字]         [角标]
```

#### 样式改进
- 字体大小：11px → 12px（更清晰）
- 字重：700 → 600（更柔和）
- 预览文字添加 `flex: 1` 和 `min-width: 0`，确保正确截断
- 底部行改为 `justify-content: space-between`，角标固定在右侧

#### 未读状态突出显示
当有未读消息时：
- **时间戳**: 
  - 颜色变为绿色 (#25d366)
  - 字重增加到 600
  - 新增 `.sidebar-item__time--unread` 类
  
- **预览文字**: 
  - 颜色从灰色 (#667781) 变为深色 (#111b21)
  - 字重增加到 500
  - 新增 `.sidebar-item__preview--unread` 类

### 3. 会话列表项样式调整

**边框改进**:
- 移除圆角：border-radius: 12px → 0
- 移除外边距：margin: 4px 8px → 0
- 取消宽度计算：width: calc(100% - 16px) → 100%
- 添加底部分隔线：border-bottom: 1px solid #f0f2f5
- 背景：透明 → 白色

**效果**: 会话列表项现在是全宽无圆角，更贴近 WhatsApp 的扁平化设计，每一项之间有清晰的分隔线。

### 4. 标题文字样式

**选中状态**:
- 选中的会话标题字重增加到 600
- 未选中保持 500

### 5. 空状态标题优化

当没有选择聊天时显示的 "EdgeChat" 标题：
- 字体：从衬线斜体改为无衬线
- 字重：400 → 300（更轻盈）
- 字号：28px → 32px（更大）
- 字距：0.02em → 0.5px
- 颜色：#111b21 → #8696a0（更淡的灰色）

**效果**: 去掉了花哨的衬线和斜体，采用简洁的无衬线字体，更符合 WhatsApp 的现代简约风格。

### 6. 侧边栏顶部优化

- 背景色统一为 #f0f2f5
- 添加底部边框分隔
- 添加按钮容器 `.sidebar-header-actions` 便于扩展

## WhatsApp 风格特点总结

### 未读角标设计原则
1. **位置**: 在底部行的最右侧，独立显示
2. **颜色**: WhatsApp 绿 (#25d366)
3. **大小**: 清晰可读但不过分突出（20px 高度，12px 字体）
4. **圆形**: 完全圆角 (border-radius: 999px)

### 未读消息视觉层次
1. **最高优先级**: 绿色角标数字
2. **次高优先级**: 绿色加粗时间戳
3. **中等优先级**: 加粗的深色预览文字
4. **低优先级**: 标题保持一致（不因未读而改变）

### 整体布局特点
1. **扁平化**: 无圆角，全宽列表项
2. **清晰分隔**: 淡色分隔线而非空白
3. **简洁配色**: 白色/浅灰/深灰，绿色点缀
4. **低调标题**: 深色而非绿色

## 视觉效果对比

### 改动前
- 绿色大标题，视觉重心过高
- 未读角标紧贴预览文字，容易被忽略
- 圆角列表项，有间距，视觉不够紧凑
- 时间戳始终灰色，无法突出未读状态

### 改动后
- 深色低调标题，让内容成为焦点
- 未读角标独立在右侧，一目了然
- 无圆角全宽列表，信息密度更高
- 有未读时绿色时间戳和角标形成视觉对
- 预览文字在有未读时变深变粗，更易注意

## 用户体验提升

1. **更快识别未读消息**: 绿色角标在右侧独立显示，扫一眼就能看到
2. **清晰的视觉层次**: 未读消息通过颜色和字重形成三层强调
3. **符合习惯**: 与 WhatsApp 的设计语言保持一致，降低学习成本
4. **更高的信息密度**: 无圆角无间距的列表项能显示更多会话

## 技术实现

### Vue 模板绑定
```vue
<!-- 时间戳根据未读状态动态添加类 -->
<span class="sidebar-label sidebar-item__time" 
      :class="{ 'sidebar-item__time--unread': item.unreadCount > 0 }">

<!-- 预览文字根据未读状态动态添加类 -->
<p class="sidebar-label sidebar-item__preview" 
   :class="{ 'sidebar-item__preview--unread': item.unreadCount > 0 }">

<!-- 角标仅在有未读时显示 -->
<span v-if="item.unreadCount > 0" class="sidebar-unread-badge">
```

### CSS 样式类
```css
/* 未读时间戳 - 绿色加粗 */
.sidebar-item__time--unread {
  color: #25d366;
  font-weight: 600;
}

/* 未读预览 - 深色加粗 */
.sidebar-item__preview--unread {
  color: #111b21;
  font-weight: 500;
}

/* 角标 - 绿色圆形 */
.sidebar-unread-badge {
  background: #25d366;
  border-radius: 999px;
  /* ... */
}
```
